# K-NanoMFA v1.5.1 Manual Addendum

Version 1.5.1 is a maintenance release that preserves all v1.5 functions and corrects the layout of the MFA mode-selection cards shown in the Study and material workspace.

## Corrected interface behavior

- The title, summary sentence, and mini-flow box in each model card are now placed on independent grid rows.
- Long text wraps within the card instead of colliding with neighboring content.
- The three model cards change from 3 columns to 2 columns and then to 1 column as the screen narrows.
- The mini-flow badge now remains within the card boundary and no longer crowds the descriptive text.

## Scientific scope

No equations, parameter handling, product-informed calculations, custom-geography calculations, nanoform accounting, or advanced MFA calculations were changed in v1.5.1. This release corrects layout behavior only.
